FE 개발 환경 온보딩 가이드: `docs/env_set/fe.md`.

**Why:** 주니어 개발자가 처음부터 따라 하면 FE 환경을 완성할 수 있는 온보딩 문서로 요청됨.

**How to apply:** 환경 설정 문서 갱신 요청 시 이 파일을 수정. 작성 규칙:
- 각 단계에 "왜 하는지 / 확인 방법 / 안 될 때(트러블슈팅)" 포함(CoT 방식)
- 명령어/설정은 실제 파일 인용만, 추측 금지. 미확인 항목은 명시(예: 클론 주소, 테스트 러너 미설치)
- 한국어 설명, 영어 식별자, 세미콜론 없는 코드 예시
